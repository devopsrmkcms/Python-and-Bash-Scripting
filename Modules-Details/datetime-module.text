# ✅ 🔥 Important `datetime` One‑Liners

***

## ✅ 1. Current date & time

```python
from datetime import datetime
print(datetime.now())
```

***

## ✅ 2. Only date

```python
print(datetime.now().date())
```

***

## ✅ 3. Only time

```python
print(datetime.now().time())
```

***

## ✅ 4. Format date (MOST IMPORTANT 🔥)

```python
print(datetime.now().strftime("%Y-%m-%d %H:%M:%S"))
```

👉 Output:

```
2026-05-24 20:45:00
```

***

## ✅ 5. Custom format

```python
print(datetime.now().strftime("%d-%m-%Y"))
```

👉 Output:

```
24-05-2026
```

***

## ✅ 6. Get individual values

```python
now = datetime.now()
print(now.year, now.month, now.day)
```

***

## ✅ 7. Add time

```python
from datetime import timedelta

print(datetime.now() + timedelta(days=1))
```

***

## ✅ 8. Subtract time

```python
print(datetime.now() - timedelta(hours=5))
```

***

## ✅ 9. Difference between two times

```python
start = datetime.now()
end = datetime.now()

print(end - start)
```

***

## ✅ 10. Convert string → datetime

```python
datetime.strptime("2026-05-24", "%Y-%m-%d")
```

***

## ✅ 11. Convert datetime → string

```python
datetime.now().strftime("%Y-%m-%d")
```

***

## ✅ 12. Timestamp (epoch)

```python
datetime.now().timestamp()
```

***

# ✅ 🔥 Small DevOps Programs

***

## ✅ 1. Log with timestamp (VERY IMPORTANT)

```python
from datetime import datetime

print(f"[{datetime.now().strftime('%Y-%m-%d %H:%M:%S')}] Service started")
```

👉 Used in:

* logging
* monitoring scripts

***

## ✅ 2. File backup naming script

```python
from datetime import datetime

filename = f"backup_{datetime.now().strftime('%Y%m%d_%H%M%S')}.tar.gz"
print(filename)
```

👉 Output:

```
backup_20260524_204500.tar.gz
```

***

## ✅ 3. Check if script runs at specific time

```python
from datetime import datetime

now = datetime.now()

if now.hour == 2:
    print("Run backup job")
```

***

## ✅ 4. Calculate downtime duration

```python
from datetime import datetime

start = datetime.now()
# simulate delay
end = datetime.now()

print(f"Downtime: {end - start}")
```

***

## ✅ 5. Alert if file older than X hours

```python
from datetime import datetime, timedelta
import os

file = "test.txt"

file_time = datetime.fromtimestamp(os.path.getmtime(file))

if datetime.now() - file_time > timedelta(hours=1):
    print("File is old - alert!")
```

👉 Used in:

* log monitoring
* stale file detection

***

## ✅ 6. Expiry check (token/session)

```python
from datetime import datetime, timedelta

expiry = datetime.now() + timedelta(minutes=30)

if datetime.now() > expiry:
    print("Session expired")
```

***

## ✅ 7. Cron-like delay check

```python
from datetime import datetime

print(f"Current time: {datetime.now().strftime('%H:%M')}")
```

# 🧠 DevOps Use Cases (IMPORTANT 🔥)

***

## ✅ Where you use `datetime`

| Use case       | Example              |
| -------------- | -------------------- |
| Logging        | timestamps in logs   |
| Backup scripts | timestamp file names |
| Monitoring     | alert if old logs    |
| Scheduling     | run at specific hour |
| Expiry checks  | tokens, sessions     |
| SLA tracking   | downtime calculation |

***

# ✅ ⚡ `datetime` vs `time`

| Module     | Use                           |
| ---------- | ----------------------------- |
| `time`     | delays, sleep                 |
| `datetime` | date + time logic, formatting |

***

# ✅ 🧠 One-line understanding

> `datetime` is used to **track, format, compare, and manipulate date & time in scripts**
