# ✅ 🔥 Important `time` module One‑Liners

***

## ✅ 1. Sleep (pause execution)

```python
import time
time.sleep(5)   # wait 5 seconds
```

👉 Used in:

* retries
* monitoring loops

***

## ✅ 2. Current timestamp (epoch)

```python
import time
print(time.time())
```

👉 Output:

```
1710000000.123
```

👉 Used in:

* logging
* time difference calculation

***

## ✅ 3. Human readable time

```python
print(time.ctime())
```

👉 Output:

```
Sun May 26 10:30:12 2026
```

***

## ✅ 4. Format time (very important)

```python
print(time.strftime("%Y-%m-%d %H:%M:%S"))
```

👉 Output:

```
2026-05-24 20:30:00
```

👉 Used in:

* logs
* file naming

***

## ✅ 5. Convert timestamp → readable

```python
print(time.ctime(time.time()))
```

***

## ✅ 6. Measure execution time

```python
start = time.time()
time.sleep(2)
end = time.time()

print(f"Time taken: {end - start:.2f} seconds")
```

***

## ✅ 7. Infinite loop (monitoring pattern)

```python
while True:
    print("Running...")
    time.sleep(5)
```

👉 Used in:

* monitoring scripts
* daemons

***

## ✅ 8. Delay retry logic

```python
for i in range(3):
    print("Retrying...")
    time.sleep(2)
```

***

## ✅ 9. Convert seconds → readable

```python
print(time.strftime("%H:%M:%S", time.gmtime(3661)))
```

👉 Output:

```
01:01:01
```

***

## ✅ 10. Get localtime object

```python
t = time.localtime()
print(t.tm_year, t.tm_hour)
```

# ✅ 🔥 Small Practical Programs

***

## ✅ 1. Simple timer script

```python
import time

print("Starting timer...")
start = time.time()

time.sleep(3)

end = time.time()

print(f"Total execution time: {end - start:.2f} seconds")
```

***

## ✅ 2. Monitoring loop (DevOps style)

```python
import time

while True:
    print(f"Checking system at: {time.strftime('%Y-%m-%d %H:%M:%S')}")
    time.sleep(5)
```

***

## ✅ 3. Retry mechanism (important in APIs)

```python
import time

for i in range(3):
    print(f"Attempt {i+1}")
    time.sleep(2)

print("Done retrying")
```

***

## ✅ 4. Countdown timer

```python
import time

for i in range(5, 0, -1):
    print(i)
    time.sleep(1)

print("Time's up!")
```

***

## ✅ 5. Log timestamp generator

```python
import time

print(f"[{time.strftime('%Y-%m-%d %H:%M:%S')}] Service started")
```

***

## ✅ 6. Script timeout example

```python
import time

start = time.time()
timeout = 5

while True:
    if time.time() - start > timeout:
        print("Timeout reached")
        break

    print("Running...")
    time.sleep(1)
```
